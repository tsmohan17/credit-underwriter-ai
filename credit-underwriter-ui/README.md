# Credit Underwriter UI

The **Credit Underwriter Client Dashboard** is a standalone Angular frontend web application. It provides an intuitive, responsive interface for applicants to apply for loans, underwriters to monitor risk assessments, and users to chat with the AI credit advisor.

---

## 🛠️ Tech Stack & Features
* **Angular v18+** (using Standalone Components, zero NgModule boilerplate)
* **TypeScript** & **RxJS** (Reactive extensions for event streams)
* **Reactive Forms** (Input binding, state management, validations)
* **Tailored Dark UI Theme** (custom layout with custom animations and status indicators)
* **Real-time Status Polling** (uses RxJS `switchMap` and `interval` to automatically fetch status updates every 3 seconds)

---

## 💻 Pages & Features

### 1. Underwriter Dashboard
Displays a real-time list of all loan applications. Clicking a row slides open the **Risk Analysis Panel**, rendering the credit rating and the detailed evaluation summary generated by the backend AI service.

### 2. Loan Application Form
A reactive input form capturing personal and financial details. Validations are enforced on:
* Email patterns.
* Phone number length (exactly 10 digits).
* PAN card formatting (Indian tax identifier).
* Positive values for income and loan requests.

### 3. AI Credit Advisor
A mock/real LLM conversational chat portal helping users find tips on how to decrease credit risk, manage DTI ratios, and improve eligibility.

---

## 🚀 Setup & Execution
1. Ensure both backend services are running.
2. Install packages:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run start
   ```
4. Navigate to `http://localhost:4200/` in your browser.
